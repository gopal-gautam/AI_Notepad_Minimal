# Contributing

Thanks for your interest in improving Minimal Notepad AI.

## Getting Started

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make focused changes.
4. Build and test locally.
5. Open a pull request with a clear description.

## Development Setup

1. Install .NET 8 SDK.
2. Open `Minimal_Notepad_AI.slnx` in Visual Studio 2022 or newer.
3. Run `dotnet build` from the repository root.
4. Run `dotnet run --project Minimal_Notepad_AI` to launch the app.

## Coding Guidelines

- Keep changes small and easy to review.
- Follow existing project naming and style conventions.
- Prefer clear method names and short, focused methods.
- Do not commit generated files from `bin` or `obj`.

## Pull Request Checklist

- Code builds successfully.
- New behavior is documented in `README.md` when needed.
- Existing behavior is not broken.
- No unrelated formatting-only changes.

## Reporting Issues

When opening an issue, include:

- What happened.
- What you expected.
- Steps to reproduce.
- Environment details (Windows version, .NET version, provider used).
